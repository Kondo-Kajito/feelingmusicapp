# MoodTunes AI 🎵

「今の気持ち」に寄り添い、埋もれている名曲とあなたを繋ぐAI音楽レコメンドチャットアプリです。

---

## （１）プロジェクト概要

* **背景（ハッカソンテーマ）：** 日常にユーモワを
* **課題：** 世界にはまだ日の目を浴びていない、埋もれている名曲が数多く存在します。しかし、現在の最適化されたアルゴリズムの中では、どうしても「いつものお気に入り」に閉じこもりがちになり、新しい名曲と出会う機会が失われています。
* **解決策：** 日常の「今の気分」や「天気」をAIが汲み取り、クスッと笑えるようなユーモアあふれるDJメッセージと共に、少し意外で魅力的な「隠れた名曲」をレコメンドします。いつも通りの毎日に、音楽を通じた心地よい驚き（ユーモア）を届けます。

---

## （２）使用技術

フロントエンドとバックエンドを明確に分離し、APIでシームレスに連携させたフルスタックな構成を採用しています。

### フロントエンド

* **React (Vite):** コンポーネントベースの高速なSPA（シングルページアプリケーション）開発
* **CSS3:** LINE風の右寄せチャットレイアウト、画面を圧迫しないコンパクトなダークテーマUIの実現
* **HTML5 Audio API:** 楽曲プレビューのシームレスなスムーズ再生

### バックエンド

* **Python (FastAPI):** 高速かつ軽量な非同期APIサーバーの構築
* **Gemini 2.0 Flash (Google Generative AI):** ユーザーのテキストから感情やコンテキストを高度に分析
* **iTunes Search API:** 生成されたキーワードを元に、実在する楽曲データを動的に検索・取得

---

## （３）主な機能

1. **AI感情分析チャット:** ユーザーの気分や環境メタデータに応じた、温かくユーモラスなDJ風返答メッセージの生成。
2. **LINE風メッセージUI:** 自分の発言は右側、AIは左側に表示され、文字量に応じてピッタリ変化・自動改行される心地よいチャット画面。
3. **楽曲カード＆ミニプレイヤー:** チャット内にカード型で表示される楽曲をその場で試聴可能。画面を切り替えても再生が途切れない永続ミニプレイヤーを搭載。
4. **ライブラリとお気に入りプレイリスト:** 気に入った曲に「いいね」をつけたり、カスタムプレイリストを作成して楽曲を保存・管理する機能。
5. **プロフィールパーソナライズ:** 好きな音楽ジャンル、年代、現在の天気をあらかじめ設定し、レコメンドの精度を向上させる環境設定機能。

---

## （４）使い方

1. **プロフィールの設定:** 「プロフィール」タブで、あなたの好きな音楽ジャンル（J-POPなど）、年代（90'sなど）、現在の天気を設定します。
2. **気持ちを話す:** 「ホーム」タブのチャット入力欄に、「仕事で疲れた」「明日が楽しみでワクワクする！」など、今のリアルな気持ちを入力して送信します。
3. **音楽と出会う:** AIコンシェルジュがあなたの心に寄り添うメッセージと共に、日常のスパイスとなる隠れた名曲をカード形式で提案してくれます。
4. **試聴と保存:** 楽曲カードをタップして音楽を試聴。気に入ったら「いいね」でライブラリに保存したり、プラスボタンからプレイリストを作成して追加できます。

## （５）セットアップ手順

※バックエンドAPI（Python/FastAPI）は既にクラウド環境（Renderなど）にデプロイされているため、ローカル環境ではフロントエンドを起動するだけで、AIチャットや音楽再生を含む全機能がそのまま動作します。

### フロントエンドの起動手順

1. **リポジトリのクローン**

   ```bash
   git clone <あなたのリポジトリのURL>
   cd <プロジェクトのディレクトリ名>

2. **依存パッケージのインストール**

    ※ルートディレクトリ（package.json がある階層）で以下を実行します。
    npm install

3. **開発サーバーの起動**

    npm run dev

4. **ブラウザで確認** ターミナルに表示されるURL（ http://localhost:5173 など）にアクセスするとアプリが起動します。
